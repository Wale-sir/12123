# 交警短信监控器

一个Android应用，用于监控手机短信，当收到交警相关短信时自动播放铃声提醒，并支持后台自启动。

## 功能特性

- 📱 **短信监控**: 实时监控收到的短信
- 🚔 **交警识别**: 智能识别交警相关短信（基于关键词和号码）
- 🔊 **铃声提醒**: 收到交警短信时自动播放提醒铃声
- 📳 **震动提醒**: 配合铃声的震动提醒
- 🔄 **后台自启动**: 开机自动启动监控服务
- 🛡️ **前台服务**: 使用前台服务确保不被系统杀死

## 技术架构

- **MainActivity**: 主界面，提供启动/停止服务的控制
- **SMSMonitorService**: 前台服务，保持应用在后台运行
- **SMSReceiver**: 广播接收器，监听短信接收事件
- **BootReceiver**: 开机启动接收器，实现自启动
- **PoliceSMSChecker**: 交警短信识别工具
- **SoundPlayer**: 声音播放和震动工具

## 权限要求

应用需要以下权限才能正常工作：

- `RECEIVE_SMS`: 接收短信
- `READ_SMS`: 读取短信内容
- `WAKE_LOCK`: 保持设备唤醒
- `FOREGROUND_SERVICE`: 前台服务
- `RECEIVE_BOOT_COMPLETED`: 开机启动
- `VIBRATE`: 震动提醒

## 安装说明

1. 使用Android Studio打开项目
2. 连接Android设备或启动模拟器
3. 点击运行按钮安装应用
4. 首次运行时授予必要权限
5. 点击"启动监控服务"开始监控

## 使用说明

1. **启动服务**: 点击"启动监控服务"按钮
2. **停止服务**: 点击"停止监控服务"按钮
3. **自动启动**: 应用会在开机后自动启动监控服务
4. **短信提醒**: 收到交警短信时会自动播放铃声和震动

## 交警短信识别规则

### 号码识别
- 12123 (交管12123)
- 122 (交通事故报警)
- 110 (报警服务)
- 10086/10000/10010 (运营商服务)

### 关键词识别
- 交通相关: 交警、交通、违章、超速、闯红灯
- 证件相关: 驾驶证、行驶证
- 处罚相关: 扣分、罚款、违法、事故
- 通知相关: 通知、提醒、警告、处理
